How to import your .ics file

After downloading the .ics file, follow the steps for your calendar app:

Google Calendar

  1. Open calendar.google.com.
  2. In the left sidebar, click the + next to Other calendars → Create new calendar. Name it something like Sun & Seasons 2026.
  3. Go to Settings (gear icon) → Import & export → Import.
  4. Choose your .ics file and select the new calendar you just created.

To remove: In Settings, find the calendar under Settings for my calendars and click Delete calendar. All events disappear at once.

Apple Calendar (macOS / iOS)

  1. Double-click the .ics file (macOS) or open it from Files (iOS).
  2. When prompted, create a new calendar for the import rather than adding to an existing one.

To remove: Right-click the calendar in the sidebar and choose Delete Calendar.

Outlook (Desktop)

  1. Open Outlook and go to File → Open & Export → Import/Export.
  2. Choose Import an iCalendar (.ics) or vCalendar file and select your file.
  3. When asked, choose Import (not Open) and create a new calendar folder first if you want easy removal later.

Outlook on the Web

  1. Open outlook.live.com.
  2. Click the calendar icon, then Add calendar → Upload from file.
  3. Select your .ics file and choose a calendar to import into (create a new one if desired).

Removing the calendar

Because the .ics file is a one-time import (not a live subscription), the easiest way to remove all events is to delete the entire calendar you imported into. That's why we recommend creating a dedicated calendar (e.g. "Sun & Seasons 2026") just for this import.

If you imported into your main calendar, you would need to delete each event individually.
